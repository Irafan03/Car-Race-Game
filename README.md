# Car Race Game

Mini-jeu web en **HTML, CSS et JavaScript** où le joueur contrôle une voiture et doit éviter des obstacles.  
Une **animation d’explosion** apparaît en cas de collision.

## Fonctionnalités

- Déplacement de la voiture à gauche/droite  
- Obstacles générés automatiquement 
- Détection de collision en temps réel 

## Technologies utilisées

- **HTML5** – structure du jeu  
- **CSS3** – design et animations  
- **JavaScript (ES6+)** – logique du jeu, collisions, DOM
